# libraryapi
An API for library: books, person, and movies.

# How to make it work

## Run the application

### Command to start the application
Run command:
```cmd
mvn spring-boot:run
```

### Check the metadata
And open brower at `http://localhost:9010/apis/v1/$metadata`.
